# persona substrate

Decision-making substrate for the `persona` agent in
`.minions/programs/research.md`. The persona answers research questions
the way the maintainer would, grounding its answers in the files under
`telos/` and `memory/` here.

These files live in a **public** repository, so they hold **only generic,
publishable guidance** — engineering trade-offs, partio product
priorities, the quality bar, defer-domains, and working preferences, with
no personal data of any kind.

Curate them by hand and review changes in a normal PR, keeping that bar
in mind.
